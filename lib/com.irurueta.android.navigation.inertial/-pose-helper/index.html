PoseHelper

object PoseHelper
Content copied to clipboard

Converts SensorEvent values into a Quaternion representing a 3D rotation that represents the device attitude and a 3D point that represents the device relative translation. Attitude is absolute respect to Earth and translation is relative to an arbitrary starting point. Optionally, delta values respect last processed SensorEvent can also be obtained.
