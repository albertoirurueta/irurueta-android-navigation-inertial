fun convertToNED(context: Context, values: FloatArray, result: Quaternion, displayOrientationResult: Quaternion? = null): Double?

Converts an array of values contained in a SensorEvent into a 3D rotation expressed in NED (North, Easth, Down) system coordinates.

Return

accuracy of heading attitude angle expressed in radians or null if not available.

Parameters

context

Android context.

values

array of values to be converted.

result

instance where converted attitude will be stored.

displayOrientationResult

instance containing a quaternion indicating display orientation as a rotation on z-axis (yaw angle). If provided, this is meant to be reused.

Throws

kotlin.IllegalArgumentException

if provided values array does not have at least MIN_LENGTH elements.
