AttitudeECEFAbsolutePoseProcessor

class AttitudeECEFAbsolutePoseProcessor(initialLocation: Location, initialVelocity: NEDVelocity, estimatePoseTransformation: Boolean, processorListener: BaseECEFAbsolutePoseProcessor.OnProcessedListener?) : BaseECEFAbsolutePoseProcessor
Content copied to clipboard

Estimates absolute pose expressed in ECEF coordinates. This class estimates attitude by using Android's absolute attitude sensor. Accelerometer and gyroscope are then taken into account to update device position.
