const val DEFAULT_GYROSCOPE_NOISE_PSD: Double = 1.0E-7

Default variance of the gyroscope output per Hz (or variance at 1Hz). This is equivalent to the gyroscope PSD (Power Spectral Density) that can be obtained during calibration or with noise estimators. This is a typical value that can be used for non-calibrated devices.
