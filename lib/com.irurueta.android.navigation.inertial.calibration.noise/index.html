Estimates accelerometer noise. This estimator takes a given number of accelerometer measurements during a given duration of time to estimate accelerometer measurements average, standard deviation and variance, as well as average time interval between measurements. To be able to measure accelerometer noise, device should remain static so that average accelerometer measurements are constant and their standard deviations reflect actual sensor noise.
