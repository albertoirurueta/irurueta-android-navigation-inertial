DEFAULT_ROBUST_THRESHOLD_FACTOR
Link copied to clipboard
const val DEFAULT_ROBUST_THRESHOLD_FACTOR: Double = 3.0
Content copied to clipboard

Default factor to be used respect estimated base noise level to consider a measurement an outlier when using robust calibration methods. By default this is 3.0 times magnetometerBaseNoiseLevel, which considering the noise level as the standard deviation of a Gaussian distribution, should account for 99% of the cases. Any measurement having an error greater than that in the estimated solution, will be considered an outlier and be discarded.
