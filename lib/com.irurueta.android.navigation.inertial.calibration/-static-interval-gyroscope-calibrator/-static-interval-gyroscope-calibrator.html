isGyroscopeGroundTruthInitialBias

true if estimated gyroscope bias is assumed to be the true value, flase if estimated bias is assumed to be only an initial guess. When gyroscopeSensorType is GyroscopeSensorType.GYROSCOPE, bias guess is zero, otherwise when it is GyroscopeSensorType.GYROSCOPE_UNCALIBRATED, bias guess is the device calibrated values.
